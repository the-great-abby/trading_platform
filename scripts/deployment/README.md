# Deployment Scripts

This directory contains scripts for deploying and activating the trading system.

## Scripts

- `deploy_optimized_system.py` - Deploy the optimized trading system
- `activate_optimized_live_trading.py` - Activate live trading with optimized settings

## Usage

```bash
# Deploy system
python scripts/deployment/deploy_optimized_system.py

# Activate live trading
python scripts/deployment/activate_optimized_live_trading.py
```
