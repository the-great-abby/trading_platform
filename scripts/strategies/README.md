# Strategy Management Scripts

This directory contains scripts for managing and selecting trading strategies.

## Scripts

- `automated_strategy_selector.py` - Automated strategy selection based on market conditions
- `hybrid_advanced_strategy.py` - Advanced hybrid strategy implementation
- `ultra_advanced_strategy.py` - Ultra-advanced strategy with multi-factor analysis

## Usage

```bash
# Run automated strategy selector
python scripts/strategies/automated_strategy_selector.py

# Run hybrid strategy
python scripts/strategies/hybrid_advanced_strategy.py
```
